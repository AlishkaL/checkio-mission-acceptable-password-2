Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę kontynuować rozwiązanie z podpowiedzi do poprzedniej misji z serii serii. 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    
    return cond1;
}

W tej misji należy dodać do tego rozwiązania sprawdzanie, czy hasło zawiera co najmniej jedną cyfrę.

Od jakiego instrumentu powinienem zacząć?

Przede wszystkim proszę zapoznać się z metodą wyrażeń regularnych test(). Testuje ona dopasowanie w ciągu znaków i zwraca true, jeśli zostanie znalezione. 

cond2 = ... .test(password);

Ok. Ale czego ja szukam?

Mówi się, że password musi mieć co najmniej jedną cyfrę. Więc jako wzór musi Pan użyć tak zwanego digit metacharacter. 

cond2 = /\d/.test(password);

Aby rozwiązać zadanie, muszę spełnić oba warunki true. Jak to zrobić?

Proszę użyć metody includes() po instrukcji return z tablicą obu warunków. 

return [cond1, cond2].includes(...);

Aby przejść testy, każdy element musi zawierać true. Proszę więc sprawdzić, czy tablica zawiera false i odwrócić wynik, ponieważ NIE musi ona zawierać false.

Jak więc powinno wyglądać moje ostateczne rozwiązanie dla tej misji?

Oto ono: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    const cond2 = /\d/.test(password);
    
    return!([cond1, cond2].includes(false));
}

Jasne, proszę łapać!

Ten skrypt nie deklaruje zmiennych, używa wyrażenia regularnego do sprawdzania obecności więcej niż 6 znaków i używa operatora AND (&&). 

function isAcceptablePassword(s) {
    return /.{7,}/.test(s) && /\d/.test(s);
    }
