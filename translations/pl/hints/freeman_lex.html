Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę kontynuować rozwiązanie z podpowiedzi do poprzedniej misji z serii serii. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password)> 6

    return cond1

W tej misji należy dodać do tego rozwiązania sprawdzanie, czy hasło zawiera co najmniej jedną cyfrę.

Od jakiego instrumentu powinienem zacząć?

Przede wszystkim proszę zapoznać się z funkcją any(). Zwraca ona True jeśli którykolwiek z elementów sekwencji wewnątrz jest True. 

cond2 = any(...)

Ok. Ale jak dokładnie sprawdzić, czy znak jest cyfrą?

Proszę użyć metody string str.isdigit(). Zwraca ona True, jeśli znak jest cyfrą.

Jest to sprawdzanie jednego znaku. Jak sprawdzić wszystkie znaki w ciągu wejściowym?

Proszę użyć funkcji map() jako argumentu funkcji any(): 

cond2 = any(map(str.isdigit, password))

Aby rozwiązać zadanie, muszę spełnić oba warunki True. Jak to zrobić?

Proszę użyć funkcji all() po instrukcji return z sekwencją obu warunków. 

return all([cond1, cond2])

Jak więc powinno wyglądać moje ostateczne rozwiązanie dla tej misji?

Oto ono: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password)> 6
    cond2 = any(map(str.isdigit, password))

    return all([cond1, cond2])

Proszę wyszukać znak za pomocą wyrażenia regularnego z wcześniej zaimportowanego modułu re. Proszę użyć funkcji bool(), aby przekształcić wynik wyszukiwania w wartość logiczną. Proszę użyć wyrażenia lambda, aby uzyskać kompaktowy widok. 

import re

is_acceptable_password = lambda x: bool(len(x)>6 and re.search('\d', x))

Proszę znaleźć przecięcie zestawu wejściowych ciągów znaków i zestawu cyfr oraz przekształcić wynik w wartość boolowską. 

is_acceptable_password = lambda x: len(x)> 6 and bool(set(x) & set("0123456789"))
