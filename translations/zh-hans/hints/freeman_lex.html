我应该从什么仪器开始？

首先，阅读关于any() 函数。如果序列中的任何元素是True ，它将返回True 。 

cond2 = any(...)

好的。但究竟如何检查该字符是否为数字？

使用字符串方法str.isdigit() 。它返回True ，如果字符是一个数字。

这只是一个字符的检查。如何检查输入字符串中的所有字符？

使用函数map() 作为函数any() 的参数： 

cond2 = any(map(str.isdigit, password))

我需要两个条件都是True ，以解决这个任务。如何做到这一点？

在返回语句后使用函数all() ，并附上两个条件的序列。 

return all([cond1, cond2])

那么我对这项任务的最终解决方案应该是怎样的呢？

这就是了： 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password)> 6
    cond2 = any(map(str.isdigit, password))

    return all([cond1, cond2])

import re

is_acceptable_password = lambda x: bool(len(x)>6 and re.search('\d', x))
