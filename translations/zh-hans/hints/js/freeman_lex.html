我应该从什么仪器开始？

首先，阅读关于正则表达式的test() 方法。它测试一个字符串中的匹配，如果找到则返回true 。 

cond2 = ... .test(password);

好的。但我要找的是什么呢？

据说，password 必须至少有一个数字。因此，作为一个模式，你需要使用所谓的digit metacharacter 。 

cond2 = /\d/.test(password);

我需要两个条件都是true 来解决这个任务。如何做到这一点？

在有两个条件的数组的返回语句后使用方法includes() 。 

return [cond1, cond2].includes(...);

为了通过测试，每个元素都必须是true 。因此，你需要检查，数组是否包括false ，并将结果反转，因为你不需要它包括false 。

那么，这个任务的最终解决方案应该是什么样子的呢？

这就是了： 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length> 6;
    const cond2 = /\d/.test(password);
    
    return!([cond1, cond2].includes(false));
}
