Я уявлення не маю, як почати розв'язування цієї місії.
Давай продовжимо розв'язок з останньої підказки попередньої місії в серії. 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6

    return cond1
У цій місії тобі слід додати до розв'язку перевірку чи пароль містить хоча б одну цифру.
З якого інструменту мені варто почати?
Перш за все, прочитай про функцію any(). Вона повертає True, якщо хоч якийсь елемент в послідовності всередині є True. 

cond2 = any(...)
Добре. Але як саме перевірити, що символ є цифрою?
Використай метод рядків str.isdigit(). Він повертає True, якщо символ є цифрою.
Це перевірки одного символа. Як перевірити всі символи у вхідному рядку?
Використай функцію map() як аргумент функції any(): 

cond2 = any(map(str.isdigit, password))
Мені потрібно, щоб обидві умови були True, щоб розв'язати цю місію. Як це зробити?
Використай функцію all() після return з послідовністю з обох умов: 

return all([cond1, cond2])
То як мій остаточний розв'язок має виглядати?
Ось він: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(map(str.isdigit, password))

    return all([cond1, cond2])

Здійсни пошук символа з допомогою регулярного виразу з попередньо імпортованого модуля re. Використай функцію bool() для перетворення результату пошуку в логічне (булеве) значення. Використай lambda вираз для компактного вигляду. 

import re

is_acceptable_password = lambda x: bool(len(x)>6 and re.search('\d', x))
Знайди перетин множини вхідного рядка і множини рядка цифр, перетвори результат в логічне (булеве) значення. 

is_acceptable_password = lambda x: len(x) > 6 and bool(set(x) & set("0123456789"))
