First of all, read about any() function. It returns True if any element of sequence inside is True.

Ok. But how exactly to check if the character is a digit?

Use string method str.isdigit(). It returns True, if character is a digit.

This is one character checking. How to check all characters in input string?

Use function map() as an argument of function any(): 

cond2 = any(map(str.isdigit, password))

I need both conditions be True to solve the mission. How to do that?

Use function all() after return statement with sequence of both conditions. 

return all([cond1, cond2])

So how my final solution for this mission should look like?

Here it is: 

def is_acceptable_password(password: str) -> bool:

    cond1 = len(password) > 6
    cond2 = any(map(str.isdigit, password))

    return all([cond1, cond2])
