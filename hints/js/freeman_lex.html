In this mission you should add to this solution checking if password contains at least one digit.

What instrument should I start from?

First of all, read about regular expressions test() method. It tests for a match in a string and returns true if found. 

cond2 = ... .test(password);

Ok. But what am I looking for?

It is said, that password must have at least one digit. So as a pattern you need to use so called digit metacharacter. 

cond2 = /\d/.test(password);

I need both conditions be true to solve the mission. How to do that?

Use method includes() after return statement with array of both conditions. 

return [cond1, cond2].includes(...);

To pass tests, every element must be true. So, you need to check, if the array includes false and invert the result since you do NOT need it to include false.

So how my final solution for this mission should look like?

Here it is: 

function isAcceptablePassword(password: string): boolean {

    const cond1 = password.length > 6;
    const cond2 = /\d/.test(password);
    
    return !([cond1, cond2].includes(false));
}
